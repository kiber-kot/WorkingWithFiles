# WorkingWithFiles

Описание кода:
Ищет все файлы, в заданной директории с любой вложеностью файлов. Производит сортировку по названию файлов. Далее записывает значения всех отсортированных файлов в один файл в лексикографическом порядке.

1. Запустить класс HelloWorld
2. В аргументы конструктора JobFiles("folder"), передать название папки, где храняться файлы
3. После выполнения кода, формируется файл в корне директории проекта
